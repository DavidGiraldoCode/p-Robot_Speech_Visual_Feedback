# p-Robot_Speech_Visual_Feedback



## ⚙️ Settup

Change directory to the Python app
```bash
cd /DesktopApp
```

Create virtual environment
```bash
python3 -m venv venv
```

Activate virtual environment before installing libraries
```bash
source venv/bin/activate
```

Install
```bash
pip install numpy scipy os matplotlib numpy pyserial PySide6
```

Deactivate
```bash
deactivate
```

To run it
```bash
python3 main.py
```
